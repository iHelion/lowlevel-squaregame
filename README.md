## To start developer server at port 8080:
1. `cd front`
2. `npm i`
3. `npm start`

## To build project:
1. `cd front`
2. `npm i`
3. `npm run build`

### Features to build:
0. add `favicon.ico`
1. Enemy
2. Random movement for enemy
3. Collision detection between player and enemy
4. Shooting
5. Collision detection between bullets and others (enemy, player, edges)
6. Health and death
7. Game start and end
